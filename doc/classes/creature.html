Functions

Creature:init (creatureType, level, noItems)
Initiates a creature. Don't call this explicitly, it's called when you create a new creature.

Parameters:

  • creatureType String. The ID of the creature you want to create.
  • level Number. The level to set the creature to (optional)
  • noItems True/False. Whether to generate without items (optional)

Returns:

    The creature itself.
Creature:generate_inventory (self, source)
Generate items for a creature's inventory. Usually called when the creature first spawns

Parameters:

  • self Creature. The creature itself
  • source Table. The source of potential item lists. Defaults to the creature itself, and will frequently be either a creature type or a faction, but can be any table containing a possible_inventory, possible_death_items, or any possible_[equipslot] table. (optional)
Creature:apply_class (self, classID)
Applies a class definition to a creature, granting it that class's starting favor, spells, factions and items

Parameters:

  • self Creature. The creature itself
  • classID String. The ID of the class to apply
Creature:get_name (self, full, force)
Get the name of a creature

Parameters:

  • self Creature. The creature itself
  • full True/False. Whether to display the creature name after the proper name (optional)
  • force True/False. If set to true, will force display the name, even if the player can't see it (optional)

Returns:

    The creature's name
Creature:get_pronoun (self, ptype)
Get the pronoun of a creature

Parameters:

  • self Creature. The creature itself
  • ptype String. The pronoun type: n = nominative, p = possessive, o = objective

Returns:

    String. The pronoun.
